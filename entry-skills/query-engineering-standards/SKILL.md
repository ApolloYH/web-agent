# 工标库检索

1. 从当前对话中识别真正需要检索的对象、场景和问题。
2. 如果当前消息只是“查工标库”之类的补充指令，把前文问题改写成一个可独立理解的完整 query。
3. 每个用户轮次只调用一次 `query_engineering_standards`；仅在首次调用明确失败或完全没有有效结果时更换关键词重试。不得用常识伪造标准编号、原文或条款。
4. 根据返回证据回答，区分检索事实、归纳结论和仍需确认的条件。
