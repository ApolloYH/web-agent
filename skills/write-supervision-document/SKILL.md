# 监理文书

First identify document type. Collect department, preparer/reviewer/approver and dates, status, project period, project overview, current construction status, and optional attachments. Preserve unknown approval fields as pending rather than inventing signatures. Save outputs under `artifacts/` as Markdown and Word.
