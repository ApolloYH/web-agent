# 监理日志

Collect project name/number, date, weather, work area and content, workforce, machinery, materials, supervision activities, issues and handling, recorder, and checker. Optional fields are other events, linked confirmed hazards, and photos. Ask only for missing required data. Save the log under `artifacts/` as Markdown and Word; generate PDF when the runtime supports it.
